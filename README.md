# Creation-of-a-model-for-forecasting-stock-time-series
Создание модели для прогнозирования временных рядов в финансовой сфере

Главная цель проекта: разработать эффективные модели машинного обучения для прогнозирования направления движения (вверх/вниз) и/или прироста цены акций Apple на N шагов вперед (на один день и на одну неделю) на основе исторических данных за последние 5 лет, создать веб приложение для интуитивно понятного пользования.
